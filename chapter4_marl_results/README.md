# Chapter 4 MARL Results

本目录用于汇总毕业论文第四章的多智能体强化学习实验结果与分析草稿。

目录说明：

- `analysis/`：第四章实验分析草稿
- `module1_convergence/`：4.5.2 收敛性与训练稳定性分析相关结果
- `module2_policy/`：4.5.3 策略演化与均衡一致性分析相关结果
- `module3_effectiveness/`：4.5.4 算法有效性分析相关结果

每个模块目录中保留：

- `figures/`：论文使用图
- `tables/`：统计表
- `csv/`：原始结果数据
- `configs/`：实验配置
- `results_summary.csv`：汇总结果
- `README.md`：模块说明

说明：

- 本次上传未包含训练检查点和中间日志 `logs/`。相关目录体量较大，属于可复现实验中间产物，不利于仓库管理。
- 论文写作时，可优先使用各模块 `figures/` 与 `tables/` 中的正式结果。
