# Chapter 4 MARL Final Delivery

本目录汇总第四章 MARL 实验的最终交付内容，按 `analysis`、`code` 和 `results` 三部分组织。

## 目录说明

- `analysis/`
  - 论文可直接使用的第四章实验分析草稿。
- `code/`
  - 第四章实验所涉及的核心代码快照，包括共享环境、模块化实验脚本与运行入口。
- `results/`
  - 三个实验模块的最终结果输出，仅保留论文需要的图、表、原始 CSV、配置与说明文件，不包含训练日志和 checkpoint。

## 模块说明

- `module1_convergence`
  - 收敛性与训练稳定性分析。
- `module2_policy`
  - 策略演化与均衡一致性分析。
- `module3_effectiveness`
  - 算法有效性分析与关键模块消融。

## 说明

本交付包保留了第四章写作所需的最终结果文件与代码快照。为控制仓库体积，训练过程中的中间日志目录和 checkpoint 未纳入该目录。
