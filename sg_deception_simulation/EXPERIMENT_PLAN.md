# Chapter 3 Numerical Simulation Plan

## Governing Theory

This module now follows the updated Chapter 3 recursion:

- use standard Bayesian update
- public state is `p_t`
- public belief evolves only by `p_{t+1} = mu_t(theta1 | sigma_t)`
- attack action does not enter public-belief update
- terminal condition is `V_{T+1}^D(theta, p) = 0`
- intermediate-stage recursive PBNE is solved stage by stage by backward induction
- all `beta`, forgetting-factor, and finite-memory belief logic is removed

## Experiment 1: Payoff Comparison

Goal:

- compare the defender's cumulative expected utility over finite horizon `T`
- compare `recursive PBNE` against the `truthful disclosure baseline`
- report the comparison under two regime-matched scenarios

Scenario design:

- Scenario A: higher initial belief `p_1`, matched with the first semi-separating structure "以真乱假"
- Scenario B: lower initial belief `p_1`, matched with the second semi-separating structure "以假乱真"

Implementation flow:

1. `backward_induction(...)`
   - start from `t = T`
   - use `V_{T+1}^D = 0`
   - recursively solve the attacker mixing probability and defender continuation values
2. `forward_simulation(...)`
   - start from the chosen `p_1`
   - apply the stage-by-stage equilibrium strategy
   - update belief only through Bayesian posterior induced by the observed signal
   - aggregate the defender's cumulative expected utility over all `T` stages
3. Result export
   - `experiment1_payoff_comparison.json`
   - `experiment1_payoff_summary.csv`
   - `experiment1_stage_paths.csv`
   - `fig3_2_cumulative_defender_utility_comparison.png`

## Experiment 2: Belief Dynamics And Terminal Distribution

Goal:

- show how public belief `p_t` evolves under recursive PBNE and truthful disclosure
- show the terminal public-belief distribution generated by Monte Carlo signal-path simulation

Implementation flow:

1. `belief_trajectory(...)`
   - sample signal paths under the equilibrium mixed strategy
   - update belief only by `p_{t+1} = mu_t(theta1 | sigma_t)`
   - record reproducible single-path and average-path outputs under a fixed random seed
2. `terminal_belief_distribution(...)`
   - reroute to the same signal-path simulator
   - aggregate the terminal public belief from repeated Monte Carlo runs
3. Result export
   - `experiment2_belief_dynamics.json`
   - `experiment2_belief_trajectories.csv`
   - `experiment2_terminal_beliefs.csv`
   - `fig3_3_public_belief_trajectories.png`
   - `fig3_4_terminal_public_belief_distribution.png`
