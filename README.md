# cyberbattle-simulation-experiment

This repository is used to organize the simulation experiments for the thesis on cyber deception defense and adversarial decision making.

## Current Scope

- `sg_deception_simulation/`:  multi-stage signaling game deception simulation, including experiment code, generated results, figures, and analysis notes.

- `sg_deception_marl/`:  MARL reproduction, optimization, and comparison experiments.

## Notes

- The  experiment is currently the first completed module in this repository.
- Generated result figures and json summaries are intentionally kept for thesis traceability.
