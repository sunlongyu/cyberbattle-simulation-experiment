# 3.4 实验仿真与分析

在本节中，为了进一步分析本章所提出改进信号博弈模型的可行性，并验证基于均衡分析得到的伪装防御策略在更真实网络攻防环境中的有效性，本文在基于 Python 的实验环境下，从理论数值仿真和 CybORG 环境仿真两个层面对所提出模型进行了实验评估。本节首先介绍实验环境和相关参数设置，然后对实验结果进行展示，并结合结果对所提策略的合理性进行分析。

## 3.4.1 实验环境设置

本节实验由两部分组成：一部分是在改进信号博弈模型下进行理论数值仿真，用于验证均衡策略的存在性及收益变化；另一部分是在 CybORG 攻防环境中进行环境仿真，用于验证该策略在更真实网络攻防链路中的实际防守效果。

理论仿真实验采用 Python 编写，在本地虚拟环境中完成；环境仿真实验同样基于 Python，并在 CybORG 所提供的网络攻防仿真框架上实现。由于本文重点关注模型与策略效果，因此本节不对底层硬件性能展开讨论；若最终论文需要补充硬件信息，可由作者根据实际实验主机配置填写。

本章改进信号博弈模型所使用的关键参数包括攻击收益、攻击成本、防御损失、攻击压力成本、欺骗情报收益、攻击者欺骗惩罚以及伪装成本等。当前理论模型的默认参数设置如下：

```text
g_a = 12, c_a = 2, l_a = 10, eta_d = 2
g_i = 8,  l_i = 8, kappa_i = 2, kappa_a = 2
c_theta1 = 2.5, c_theta2 = 1.5
```

在此基础上，本文重点设置两组典型场景：

1. 场景 A：`p = 0.65`，对应真实生产系统占比较高的情况，用于比较真实披露基线与 `PBNE-1` 生产系统伪装策略；
2. 场景 B：`p = 0.35`，对应欺骗系统相对占比更高的情况，用于比较真实披露基线与 `PBNE-2` 欺骗系统伪装策略。

在理论层中，攻击者根据均衡分析选择攻击或撤退行为，防御者根据均衡分析选择真实披露或伪装信号；在环境层中，本文将真实披露策略记为 truthful baseline，将依据均衡结果进行信号伪装与动作映射的策略分别记为 `PBNE-1` 和 `PBNE-2`。为了保证对比的清晰性，环境实验均采用相同的 CybORG thesis-specific 场景、相同的红方推进逻辑以及相同的 episode 数与步长设置，仅改变蓝方策略。

## 3.4.2 实验结果分析

### （1）理论数值仿真结果分析

首先分析改进信号博弈模型在理论层的数值实验结果。根据模型推导可知，在场景 A 中，博弈存在生产系统伪装混合均衡 `PBNE-1`；在场景 B 中，博弈存在欺骗系统伪装混合均衡 `PBNE-2`。相应结果表明：

- 在场景 A (`p = 0.65`) 下，truthful baseline 的防御者期望效用为 `-7.779`，而 `PBNE-1` 的防御者期望效用提升为 `-5.672`；同时攻击者期望效用由 `6.482` 降至 `2.338`。
- 在场景 B (`p = 0.35`) 下，truthful baseline 的防御者期望效用为 `-4.359`，而 `PBNE-2` 的防御者期望效用提升为 `-3.409`；同时攻击者期望效用由 `3.632` 降至 `1.238`。

这一结果说明，在改进后的效用函数下，依据完美贝叶斯均衡得到的伪装策略能够在理论层显著降低攻击者收益，并在总体上改善防御者效用。换言之，本章引入攻击压力成本和欺骗附加价值之后，原有“基于信号控制攻击者判断”的思路仍然成立，而且得到的均衡策略具有明确的防守意义。

### （2）环境主实验结果分析

为了进一步验证上述均衡策略在真实攻防过程中的有效性，本文在 CybORG 环境中构建了对应的 thesis-specific 场景，并采用 `12 episodes / 30 steps` 的设置进行实验。图 3.4 展示了两组主场景下蓝方平均回报的对比情况。

![图3.4 主实验蓝方平均回报对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_primary_blue_rewards.png)

从图中可以看出，在两个场景下，基于均衡分析得到的 PBNE 策略均优于真实披露基线：

- 场景 A 中，`PBNE-1` 相比 truthful baseline 使蓝方平均回报提升 `106.142`；
- 场景 B 中，`PBNE-2` 相比 truthful baseline 使蓝方平均回报提升 `69.083`。

这说明本章所提出的策略并非仅在抽象的博弈模型中有效，而是在更高保真度的网络攻防环境中同样能够产生明显的防守收益。

### （3）攻击链压制效果分析

进一步地，若仅观察平均回报，仍不足以完整说明策略为何有效。因此，本文继续分析攻击样动作数量、生产主机失陷率以及关键主机失陷率等更能反映安全效果的指标。图 3.5 给出了主实验中的安全效果指标对比。

![图3.5 主实验安全效果指标对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_primary_security_metrics.png)

从图中可以明显看出，不论是在场景 A 还是场景 B 中，PBNE 策略都降低了攻击者推进攻击链的能力：

- 场景 A 中，攻击样动作均值由 `18.000` 降至 `15.083`，生产主机失陷率由 `0.917` 降至 `0.667`，关键主机失陷率由 `0.917` 降至 `0.583`；
- 场景 B 中，攻击样动作均值由 `18.000` 降至 `15.833`，生产主机失陷率由 `0.917` 降至 `0.750`，关键主机失陷率由 `0.917` 降至 `0.750`。

上述结果表明，本章策略的主要作用机制并不是简单提高攻击者进入蜜罐的概率，而是通过信号伪装扰乱攻击者对目标类型的判断，降低其攻击推进效率，从而削弱其对高价值节点的攻击能力。

### （4）失陷结构分析

为了进一步说明策略收益的来源，图 3.6 对不同策略下生产主机、蜜罐主机和关键主机的失陷率进行了并列展示。

![图3.6 主实验失陷结构对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_compromise_breakdown.png)

从图中可以发现，PBNE 策略最稳定的效果体现在对生产主机与关键主机风险的抑制上，而蜜罐主机失陷率并不总是上升。这一现象具有重要意义：它说明本章策略的价值并不依赖于“必须把攻击者完全诱导到蜜罐上”，而在于通过信号干扰使攻击者整体更难识别和推进到真实高价值目标。

因此，在论文表述中，更准确的说法应当是：PBNE 伪装策略通过压制攻击链推进和降低关键资产暴露风险来提升防守收益，而不是单纯依靠提高蜜罐命中率获得收益。

### （5）理论与环境一致性分析

图 3.7 将理论层防御者效用增益与环境层蓝方回报增益进行了对照。

![图3.7 理论与环境方向一致性对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_theory_environment_alignment.png)

由图可见，在场景 A 与场景 B 中，理论层和环境层的增益方向均保持一致，即 PBNE 策略在理论推导下提高了防御者期望效用，在环境实验中也带来了更高的蓝方收益。这说明改进后的效用函数与环境实验在方向上具有一致性，从而能够为策略合理性提供双重支撑。

需要说明的是，理论效用与环境奖励并非严格逐项等价。这是因为理论层采用的是有效单阶段信号博弈建模，而环境层则刻画了多阶段攻击链过程。因此，本文不将二者表述为“数值完全对应”，而是强调“策略优劣方向一致、作用机制相符”。对于毕业论文而言，这种分层验证方式既保持了理论严谨性，也增强了实验可信度。

### （6）敏感性分析

为了进一步考察模型参数变化对策略效果的影响，本文对先验概率 `prior_theta1` 与信念折扣因子 `beta` 进行了环境层敏感性分析。图 3.8 和图 3.9 分别给出了对应结果。

![图3.8 prior_theta1 敏感性分析](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_sensitivity_prior_theta1.png)

![图3.9 beta 敏感性分析](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/cyborg_validation/results/figures/chapter3_sensitivity_beta.png)

从图 3.8 可以看出，当 `prior_theta1` 处于 `0.4-0.6` 区间时，PBNE 策略带来的蓝方回报提升最明显。这说明当真实系统与欺骗系统的先验不确定性较强时，信号伪装策略最容易发挥作用；而当类型分布过于偏向单一类型时，攻击者更容易形成稳定预期，从而削弱伪装策略的边际收益。

从图 3.9 可以看出，`beta` 的变化会对蓝方回报提升和攻击样动作减少幅度产生一定影响，但其影响强度小于先验概率。这说明相比于信念更新折扣速度，目标类型分布本身对策略效果的影响更为显著。

### （7）综合分析

综合理论层与环境层结果可以看出，本章改进后的信号博弈模型及其均衡防御策略具备以下特点：

1. 在理论层，PBNE 伪装策略能够提升防御者期望效用并压制攻击者收益；
2. 在环境层，PBNE 伪装策略能够显著降低攻击样动作数量、生产主机失陷率和关键主机失陷率；
3. 策略收益主要来源于攻击链压制与高价值资产保护，而非单一蜜罐命中指标的变化；
4. 理论模型与环境实验在策略优劣判断上保持方向一致，从而能够为策略合理性提供较强证据支持。

因此，可以认为本章提出的改进信号博弈模型及基于完美贝叶斯均衡的伪装策略，不仅在理论上具有可解释性，而且在更真实的网络攻防环境中同样具备较好的现实防御效果。
