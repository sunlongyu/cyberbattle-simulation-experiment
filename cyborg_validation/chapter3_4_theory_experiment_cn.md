# 3.4 实验仿真与分析

在本节中，为了验证本章所构建的多阶段信号博弈模型的可行性，并检验基于完美贝叶斯纳什均衡分析所得伪装策略的有效性，本文在 Python 环境下对所提出的博弈模型进行了数值仿真。实验部分主要包括两个方面：一是比较真实披露基线策略与适用均衡策略在不同场景下的收益差异，以验证均衡策略的有效性；二是分析关键参数变化对防御者期望效用及均衡混合概率的影响，以刻画模型的参数敏感性和部署条件。通过上述实验，可以从数值层面对前文理论推导结果进行验证，并为后续章节的进一步扩展提供基础。

## 3.4.1 实验环境设置

本节实验在本地 MacBook Pro 平台上完成，处理器为 Apple M1 八核芯片，内存为 16 GB，操作系统为 macOS 15.5，实验代码采用 Python 3.9.6 编写实现。实验程序基于前文构建的多阶段信号博弈模型开展 Monte Carlo 数值仿真，通过重复采样的方式估计不同策略下攻防双方的期望收益、攻击者后验信念以及均衡概率的变化情况。

为保证第三章实验结果与前文模型推导保持一致，实验中采用的主要参数设置如表 3-4 所示。参数取值综合考虑了真实生产系统受攻击时的损失、蜜罐系统提供的情报收益、攻击方实施攻击所需成本以及防御方实施伪装所需开销等因素，并结合本章新引入的攻击压力成本、防护收益和欺骗附加收益进行统一设定。

<center>表 3-4 博弈模型参数设置</center>

| 参数 | 含义 | 取值 |
|---|---|---|
| \(c_a\) | 攻击方执行一次攻击的成本 | 2.0 |
| \(g_a\) | 攻击方成功攻击真实生产系统获得的收益 | 12.0 |
| \(l_a\) | 真实生产系统遭攻击时防御方承受的核心损失 | 10.0 |
| \(\eta_d\) | 攻击发生时防御方承担的额外响应压力成本 | 2.0 |
| \(g_d\) | 真实生产系统成功规避攻击时防御方获得的保护收益 | 6.0 |
| \(g_i\) | 防御方诱导攻击方攻击欺骗系统时获得的基础情报收益 | 8.0 |
| \(\kappa_i\) | 防御方因成功实施欺骗获得的附加收益 | 2.0 |
| \(l_i\) | 攻击方误攻欺骗系统时遭受的基础损失 | 8.0 |
| \(\kappa_a\) | 攻击方落入欺骗环境后承担的附加惩罚 | 2.0 |
| \(c_{\theta_1}\) | 真实生产系统实施伪装的成本 | 2.5 |
| \(c_{\theta_2}\) | 蜜罐系统实施拟态伪装的成本 | 1.5 |
| \(\beta\) | 时间折扣因子 | 1.0 |

在均衡策略有效性对比实验中，本文分别设置两类具有代表性的先验场景。场景 A 取 \(p=0.65\)，表示目标集合中真实生产系统占比较高，此时根据前文均衡分析结果，适用的防御策略为 PBNE-1，即生产系统以一定概率伪装为蜜罐型信号；场景 B 取 \(p=0.35\)，表示目标集合中蜜罐系统占比较高，此时适用的防御策略为 PBNE-2，即蜜罐系统以一定概率伪装为正常系统信号。作为对比，本文设置真实披露基线策略，即生产系统始终发送正常信号，蜜罐系统始终发送蜜罐型信号。对于上述两个场景，均通过 Monte Carlo 重复仿真估计防御者期望效用、攻击者期望效用、后验信念演化路径和终局信念分布等指标。

在参数敏感性分析中，本文重点考察四类参数：先验概率 \(p\)、时间折扣因子 \(\beta\)、生产系统伪装成本 \(c_{\theta_1}\) 以及蜜罐系统伪装成本 \(c_{\theta_2}\)。其中，\(p\) 用于刻画环境中真实资产与蜜罐资产的组成结构，\(\beta\) 反映攻击者对近期信号的敏感程度，\(c_{\theta_1}\) 与 \(c_{\theta_2}\) 则分别反映两类伪装策略的实施代价。通过分析这些参数变化下防御者期望效用与均衡混合概率的变化趋势，可以进一步验证本章理论模型对环境组成和伪装成本的敏感性。

## 3.4.2 实验结果分析

### （1）均衡策略有效性对比分析

首先讨论不同先验场景下，适用 PBNE 策略与真实披露基线策略之间的收益差异。图 3-4 给出了场景 A 与场景 B 中防御者期望效用的对比结果。

![图3-4 不同场景下防御者期望效用对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/sg_deception_simulation/results/figures/fig3_1_defender_utility_comparison.png)

从图 3-4 可以看出，在场景 A（\(p=0.65\)）下，真实披露基线策略的防御者期望效用为 \(-7.7790\)，而 PBNE-1 策略的防御者期望效用提升至 \(-5.1741\)。虽然在该场景中防御者总体效用仍为负值，但均衡伪装策略能够显著降低损失水平，说明当真实生产系统在目标集合中占比较高时，生产系统以一定概率释放蜜罐型信号，能够削弱攻击者对高价值目标的锁定程度，从而改善防御方总体收益。

在场景 B（\(p=0.35\)）下，真实披露基线策略的防御者期望效用为 \(-4.3590\)，而 PBNE-2 策略的防御者期望效用提升至 \(6.1792\)。这说明在蜜罐系统占优的环境中，蜜罐通过拟态伪装成正常系统能够更有效地吸引攻击者进入欺骗环境，并为防御方带来更高的情报收益，因此 PBNE-2 在该场景下表现出更明显的优势。

从上述结果可以看出，本章所得均衡策略并非在所有环境下都以同样方式发挥作用，而是与先验环境结构具有显著相关性。当环境中真实生产系统占比较高时，生产系统伪装策略更具优势；当环境中蜜罐占比较高时，蜜罐拟态策略更具优势。这一结论与前文对两组混合均衡适用条件的分析保持一致。

### （2）攻击者后验信念演化分析

为进一步说明均衡策略发挥作用的机制，图 3-5 展示了两个场景下攻击者关于目标为真实生产系统的后验信念随阶段变化的平均轨迹。

![图3-5 攻击者后验信念随阶段变化曲线](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/sg_deception_simulation/results/figures/fig3_2_belief_trajectories.png)

从图 3-5 可以看出，在真实披露基线策略下，攻击者的后验信念变化较为平缓且趋于单调，其对目标类型的判断更容易在较少阶段内趋于稳定。相比之下，在 PBNE 伪装策略下，攻击者在不同阶段接收到具有扰动作用的信号后，其后验信念演化路径呈现出更加明显的波动性。这表明混合伪装策略并非简单改变单次收益，而是通过影响攻击者的持续推断过程，延缓其对目标真实属性的识别速度，从而改变后续攻击决策。

对于场景 A 而言，PBNE-1 策略使攻击者对生产系统的后验判断保持在较高但不完全确定的区间内，从而避免攻击者立即形成稳定攻击决策；对于场景 B 而言，PBNE-2 策略则使攻击者对生产系统的信念被持续拉低，增强了其将目标误判为真实系统或对目标真实性保持不确定的可能性。这说明前文所构建的多阶段信念更新机制能够有效刻画信号随机化对攻击者认知过程的影响。

### （3）终局信念分布分析

为了更清晰地观察伪装策略对攻击者最终判断结果的影响，图 3-6 进一步给出了场景 B 下终局时刻攻击者后验信念的分布情况。

![图3-6 场景B下终局信念分布对比](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/sg_deception_simulation/results/figures/fig3_3_final_belief_distribution.png)

由图 3-6 可以看出，在真实披露基线策略下，攻击者终局信念更多集中于相对稳定的区间，说明其对目标类型的判断更容易在观测结束时收敛；而在 PBNE-2 策略下，终局信念分布整体向较低区间偏移，且呈现出更明显的分散特征。这表明蜜罐拟态策略能够在多阶段博弈过程中持续扰动攻击者对目标真实性的识别，使其在结束时仍难以形成确定性判断。该结果进一步从分布层面验证了 PBNE-2 在低先验生产系统占比场景中的有效性。

### （4）参数敏感性分析

为了进一步考察模型参数变化对均衡结果的影响，图 3-7 给出了 \(p\)、\(\beta\)、\(c_{\theta_1}\) 和 \(c_{\theta_2}\) 四类参数的敏感性分析结果。

![图3-7 参数敏感性分析结果](/Users/SL/Documents/expproject/cyberbattle-simulation-experiment/sg_deception_simulation/results/figures/fig3_4_sensitivity_analysis.png)

首先，先验概率 \(p\) 对 PBNE-1 的影响最为显著。当 \(p\) 从 0.3 增大到 0.7 时，PBNE-1 下防御者期望效用由 \(2.7699\) 逐步下降至 \(-6.0462\)，同时生产系统伪装概率 \(\lambda_D^*\) 由 0.98 下降至 0.5537。该结果表明，随着环境中真实生产系统占比提高，攻击者更容易形成“目标更可能是真实生产系统”的先验判断，从而削弱生产系统伪装策略的边际收益。因此，在真实资产占比过高的网络环境中，防御方通过随机化信号实现有效扰动的空间会被压缩。

其次，时间折扣因子 \(\beta\) 对 PBNE-2 下防御者期望效用的影响相对较弱。在本文设置的参数区间内，\(\beta\) 从 0.2 增大到 1.4 时，防御者期望效用始终稳定在 6.08 至 6.18 之间，蜜罐拟态概率 \(\lambda_D'\) 也基本保持在 0.294 左右。这说明在当前收益结构下，\(\beta\) 的主要作用更多体现在调节攻击者对历史观测的记忆衰减速度，而不会显著改变均衡收益水平。换言之，本模型对 \(\beta\) 具有一定鲁棒性。

再次，生产系统伪装成本 \(c_{\theta_1}\) 对 PBNE-1 的影响较为直接。当 \(c_{\theta_1}\) 从 0.5 增加到 2.5 时，防御者期望效用由 \(-1.2161\) 下降至 \(-1.6499\)，攻击方在信号 \(\sigma_2\) 下的均衡攻击概率 \(\lambda_A^*\) 则由 0.9722 下降至 0.8611。该结果说明，随着生产系统伪装成本上升，防御方实施伪装的净收益减弱，而攻击方在可疑信号下的攻击积极性也会随之降低。二者共同作用，使 PBNE-1 策略的整体收益水平下降。

最后，蜜罐系统伪装成本 \(c_{\theta_2}\) 对 PBNE-2 的影响同样明显。当 \(c_{\theta_2}\) 从 0.5 增加到 2.5 时，防御者期望效用由 \(19.7379\) 下降至 \(2.6666\)，攻击方的撤退概率 \(\lambda_A'\) 由 0.95 下降至 0.75。该结果表明，蜜罐拟态成本越高，其通过伪装吸引攻击者进入欺骗环境的能力越弱，防御方能够获得的诱捕收益和情报收益也随之下降。因此，在实际部署中，应尽量控制蜜罐拟态实现的资源开销，以避免过高成本削弱策略收益。

综合上述实验结果可以看出，在与先验环境结构相匹配的适用条件下，基于 PBNE 的伪装策略能够有效提升防御者总体收益，并通过影响攻击者后验信念演化和攻击决策来实现对目标系统的保护。同时，模型对先验概率和伪装成本较为敏感，这说明在实际应用中需要结合资产构成比例和伪装实施代价，对防御策略进行针对性配置。总体而言，本节数值仿真结果验证了前文均衡分析所得策略的有效性，也说明所构建的多阶段信号博弈模型能够较好地刻画反蜜罐攻击背景下的攻防交互过程。
